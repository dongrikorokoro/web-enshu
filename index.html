code

説明

HTMLの <code> タグは，ソースコードの文字列を表示して見せるときに使います。
このコードは文中でコードを紹介するときに使います。そして、複数行のコードを乗せるときには <pre> タグの中に <code> タグを入れ、その中にコードを書きます。

使用例

まず一行の場合、

<code>&lt;p&gt;私の名前はダニエルです。&lt;/p&gt;</code>
<p>私の名前はダニエルです。</p>

そして、複数行の場合、

<pre>
    <code>
        <p>私の好きなゲームランキング</p>
        <ol>
            <li>キングダムハーツ</li>
            <li>ドラゴンクエスト</li>
            <li>ブラッドボーン</li>
        </ol>
    </code>
</pre>
<p>私の好きなゲームランキング</p>
<ol>
    <li>キングダムハーツ</li>
    <li>ドラゴンクエスト</li>
    <li>ブラッドボーン</li>
</ol>

ruby

説明

HTMLの<ruby>は、テキストにルビ（ふりがな）を振るために使います。
ルビを振るためには<ruby>と<rt>を使います。

使用例

1文字ごとにルビを振る

<ruby>
    好<rt>こう</rt>汰<rt>た</rt>
</ruby>
好こう汰た

複数文字にルビを振る

<ruby>
    好汰<rt>こうた</rt>
</ruby>
好汰こうた

samp

説明

HTMLの<samp>タグは、プログラムからの出力内容を表すために使います。

使用例

<p><samp>接続できませんでした</samp>と表示された場合、ブラウザをリロードしてください。</p>

接続できませんでしたと表示された場合、ブラウザをリロードしてください。
